    GoPxL 工具与测量 (Inspect)

    在 GoPxL 中，Inspect (检测) 页面是实现核心检测逻辑和尺寸测量的地方。在这里，您可以添加各种测量和处理工具，将原始的 2D 轮廓或 3D 点云转化为具体的测量数值（如长、宽、高、半径）以及 Pass/Fail 判定结果。

    1. 核心概念 (Core Concepts)

    在使用具体工具之前，理解以下几个 GoPxL 的核心机制至关重要。

    1.1 工具链 (Tool Chaining)

    GoPxL 的测量工具具有高度的灵活性，它们可以像积木一样连接在一起：一个工具的输出可以作为另一个工具的输入。 在中间的 Tool Diagram (工具图) 面板中，蓝色的连线清晰地展示了数据的流向。例如，您可以先用“表面滤波 (Surface Filter)”工具平滑点云，然后将滤波后的输出传递给“表面尺寸 (Surface Dimension)”工具进行测量。

    1.2 几何特征 (Geometric Features)

    很多工具除了输出具体的数值（如 10.5 mm）外，还会输出几何特征，例如点 (Point)、线 (Line)、圆 (Circle) 或平面 (Plane)。 这些几何特征本身不是测量结果，但可以传递给后端的 Feature (特征) 工具，用于进行更复杂的空间计算（例如，计算一条线和一个平面的交点，或者两个圆心之间的距离）。

    1.3 锚定 (Measurement Anchoring)

    在实际生产中，被测物体在传送带上的位置往往每次都不同（平移或旋转）。锚定 (Anchoring) 是解决这一问题的核心功能。 * 定位工具：首先添加一个工具（如 Surface Part Detection 或 Feature Intersect）来找到物体上稳定的参考点（X/Y/Z 坐标）或参考线（角度）。 * 跟随测量：将后续测量工具的 Anchor 输入（如 Anchor X, Anchor Z Angle 等）链接到上述定位工具的输出。这样，测量工具的“感兴趣区域 (Region)”就会智能地跟随物体移动和旋转，极大地提高了检测的稳定性。

    使用锚定的好处

    通过有效地使用点和线几何特征进行锚定转换，您可以确保感兴趣的特征始终位于同一局部坐标系中。这可以节省大量的设置时间，并大幅降低复杂应用的配置难度。

    1.4 感兴趣区域 (Regions)

    绝大多数工具都允许您开启 Use Region (使用区域) 选项，以限制工具只处理视野中的特定部分数据。 * 标准区域 (Standard Regions)：只支持矩形/长方体边框。 * 灵活区域 (Flexible Regions)：除了矩形，还支持圆形 (Circle)、椭圆 (Ellipse) 以及多边形 (Polygon)，部分工具甚至支持使用 3D 表面数据作为掩膜 (Mask)。

    2. 工具分类概览 (Tool Categories)

    GoPxL 提供了多达上百种内置工具，按数据类型主要分为以下几大类：

    • Profile Tools (轮廓工具)：用于处理 2D 截面数据（如测量截面面积、台阶高度、角度等）。
    • Surface Tools (表面工具)：用于处理 3D 点云或高度图数据（如测量体积、平面度、圆柱度等）。
    • Feature Tools (特征工具)：通过输入的点、线、面等几何特征，计算它们之间的尺寸（Feature Dimension）或创建新的特征（Feature Create）。
    • Array Tools (数组工具)：用于处理包含多个元素的数据集，例如将多个测量值打包，或从数组中提取特定索引的数据。
    • Script / Formula (脚本与公式)：用于编写自定义的逻辑或数学运算。

    3. 常用基础工具详解

    3.1 表面尺寸 (Surface Dimension)

    功能：快速测量 3D 物体的关键尺寸。 使用方法： 需要指定两个特征点类型（Feature 1 & Feature 2），如最大Z值 (Max Z)、中心点 (Centroid)、左边缘 (Min X) 等。工具会自动计算这两个特征点之间的距离、X/Y/Z 轴的宽度和高度。

    3.2 表面孔洞 (Surface Hole)

    功能：定位并测量 3D 数据中的圆孔。 输出结果：不仅输出孔的半径 (Radius)，还会输出孔的中心点坐标 (X, Y, Z) 以及中心点几何特征 (Center Point)。中心点几何特征经常被用来作为其他工具的锚定源。

    3.3 表面零件检测 (Surface Part Detection)

    功能：在同一视野中分离和识别多个独立的对象（通常用于连续传输带扫描）。 特点：可以基于高度阈值 (Height Threshold) 和面积过滤掉背景传送带和杂讯，将各个独立的零件分割出来，并输出零件数量 (Part Count) 以及每个零件的独立 3D 表面数据 (Part Surface)。

    3.4 表面平面度 (Surface Flatness)

    功能：评估物体表面的平整程度。 特点：工具会在指定的区域内拟合一个参考平面，然后计算实际数据点相对于该拟合平面的全局平面度（所有点的最大正偏差与最大负偏差之和）。

    3.5 测量公式 (Measurement Formula)

    功能：如果您需要将两个工具的测量结果相加、相减或进行更复杂的数学运算，可以使用此工具。 使用方法： 将其他工具的测量结果引入为变量（如 M1, M2），然后在 Formula 文本框中编写表达式，如 M2 - M1 或 max(M1, M2)，工具会输出最终的 Result 值。

    3.6 脚本工具 (Script)

    功能：当内置工具无法满足复杂的逻辑判断（如 if-else 分支）或高级数学计算时，可以使用 Python 脚本工具。 特点：内置 Python 3.8 引擎，支持读取其他工具的输出（包括点云、轮廓、测量值），执行自定义的 Python 代码，并输出新的结果。

    4. 界面操作技巧 (Tips & Tricks)

    快速编辑模式 (Quick Edit)

    当您的任务中包含大量测量工具（例如几十个），或者使用了非常复杂的工具时，每次修改参数后数据视图的刷新可能会导致界面卡顿。您可以开启界面顶部的 Quick Edit (快速编辑) 开关。在此模式下，修改参数后视图不会立即刷新，从而大幅提升配置速度。配置完成后，请务必关闭该模式以保存并应用更改。

    • 添加工具：在左侧搜索框输入工具名称，双击即可添加到工具链中，或者直接拖拽到右侧的 Diagram 面板中。
    • 复制/重命名/删除：在 Tool Diagram 中点击工具右上角的齿轮图标（操作菜单），可以快速复制 (Duplicate) 一个参数完全相同的工具，或者对其重命名以增加代码可读性（如命名为“左侧螺丝孔测量”）。
    • 固定输出 (Pinning Outputs)：在工具的 Outputs 列表中，点击图钉 (Pin) 图标，可以将该测量值固定显示在主数据视图的右侧，方便实时监控。 ```
